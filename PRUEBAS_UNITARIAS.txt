3.2 Configuracion SMTP cargada

Resultado esperado:
  True

3.3 Login SMTP sin enviar correo
Comando:
  python - <<PY
  from pathlib import Path
  import smtplib

  env = {}
  for line in Path('.env').read_text(encoding='utf-8').splitlines():
      line = line.strip()
      if not line or line.startswith('#') or '=' not in line:
          continue
      key, value = line.split('=', 1)
      env[key] = value.strip()

  host = env['SMTP_HOST']
  port = int(env.get('SMTP_PORT', '587'))
  username = env['SMTP_USERNAME']
  password = env['SMTP_PASSWORD']
  if host.lower() == 'smtp.gmail.com' and username.lower().endswith('@gmail.com'):
      password = password.replace(' ', '')

  encryption = env.get('SMTP_ENCRYPTION', 'tls').lower()
  server = smtplib.SMTP_SSL(host, port, timeout=20) if encryption == 'ssl' else smtplib.SMTP(host, port, timeout=20)
  try:
      server.ehlo()
      if encryption == 'tls':
          server.starttls()
          server.ehlo()
      server.login(username, password)
      print('SMTP login OK')
  finally:
      server.quit()
  PY

Resultado esperado:
  SMTP login OK


4. Pruebas unitarias sugeridas para backend
-------------------------------------------

4.1 email_service.smtp_configured
Caso:
  - SMTP_HOST, SMTP_USERNAME y SMTP_PASSWORD tienen valores.
Resultado esperado:
  - Retorna True.

Caso:
  - Falta SMTP_PASSWORD.
Resultado esperado:
  - Retorna False.

4.2 email_service._smtp_password
Caso:
  - Host smtp.gmail.com, username Gmail y password con espacios.
Resultado esperado:
  - Retorna password sin espacios.

Caso:
  - Host distinto a Gmail.
Resultado esperado:
  - Retorna password con strip, sin modificar espacios internos.

4.3 email_service.send_candidate_interview_email
Caso:
  - SMTP no configurado.
Resultado esperado:
  - Retorna False y no intenta enviar.

Caso:
  - SMTPAuthenticationError.
Resultado esperado:
  - Retorna False.

Caso:
  - SMTPException.
Resultado esperado:
  - Retorna False.

Caso:
  - Envio exitoso.
Resultado esperado:
  - Retorna True.

6. Pruebas de integracion manual
--------------------------------

6.1 Crear plantilla
Pasos:
  1. Abrir http://localhost:3000/dashboard/templates/new
  2. Completar titulo, puesto y descripcion.
  3. Agregar al menos 2 requisitos.
  4. Agregar al menos 2 preguntas.
  5. Guardar plantilla.

Resultado esperado:
  - Redirige a detalle de plantilla.
  - La plantilla aparece en /dashboard/templates.

6.2 Crear entrevista
Pasos:
  1. Abrir http://localhost:3000/dashboard/interviews/new
  2. Seleccionar plantilla.
  3. Escribir nombre y correo del candidato.
  4. Subir CV PDF.
  5. Crear y enviar acceso.

Resultado esperado:
  - Se crea entrevista.
  - Se analiza CV.
  - Se genera token y enlace.
  - Se intenta enviar correo por SMTP.

6.3 Acceso candidato
Pasos:
  1. Copiar enlace generado.
  2. Abrir enlace en navegador.
  3. Validar token.

Resultado esperado:
  - Carga bienvenida del candidato.
  - Muestra preguntas disponibles.

6.4 Reporte
Pasos:
  1. Completar entrevista.
  2. Abrir http://localhost:3000/dashboard/reports
  3. Entrar al reporte del candidato.

Resultado esperado:
  - Muestra score, recomendacion, skills, fortalezas, areas de mejora y respuestas evaluadas.
